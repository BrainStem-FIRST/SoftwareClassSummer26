Part 3: Day3BrainSTEMRobot
Create a Day3BrainSTEMRobot class that ties
your Arm, Lift, and a pre-existing drivetrain class together into one robot object, and handles battery
voltage for you so the individual subsystems don't have to know about it.
See Day3BrainSTEMRobotAnswerKey.java for the answer key

Class Requirements:
Instance data:
- a BatteryVoltageFilter (private - no other class needs to touch this directly)
- a MecanumDrive, called drive (public final)
- your Lift class, called lift (public final)
- your Arm class, called arm (public final)

The Day3BrainSTEMRobot class should have a constructor taking in HardwareMap, Telemetry, and
an initial Pose2d
It should:
- instantiate the BatteryVoltageFilter using hardwareMap
- instantiate drive using hardwareMap and the initial pose
- instantiate lift and arm using hardwareMap and telemetry

It should have an update function (takes 0 parameters)
- get the current battery voltage from the BatteryVoltageFilter
- call setBatteryVoltage on both lift and arm, passing in that voltage
  - IMPORTANT: you must do this BEFORE calling lift.update() and arm.update(). If you update
    the subsystems first, they'll be using last loop's stale battery voltage instead of this
    loop's
- update your drive's localizer (drive.pinpoint().update())
- call lift.update() and arm.update()



Part 4: CommandsList
Create a CommandsList class. This class will hold public static methods that all return InstantCommands.
You will be writing InstantCommands that set subsystem states/target positions (the simplest type of commands).
Each method takes in the subsystem it needs (plus any other arguments it needs) as parameters, and returns a new InstantCommand whose body calls the appropriate
setter(s) on that subsystem.

The list of functions you have to write is:
- collectorIntake - sets the Collector's intake state to INTAKE
- collectorOff - sets the Collector's intake state to OFF
- turretToCenter - sets the Turret's state to POINT_AT_ANGLE and its target angle to 0
- liftToTargetPosition - sets the Lift's target position to the position argument
- armToTargetAngle - sets the Arm's target angle to the angleRad argument
- armTo90Degrees - sets the Arm's target angle to 90 degrees (converted to radians)

See Day3CommandsListAnswerKey.java for the answer key
